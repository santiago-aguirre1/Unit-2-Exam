Windows 10 was released in 2015. Most people uses windows 10 now.


Microsoft's Windows NT operating system saw a significant update with Windows 10. It is the direct replacement for Windows 8.1, which was made available almost two years prior. July 15, 2015, saw the start of manufacturing, and on July 29, 2015, it went on sale. Downloads of Windows 10 were made possible via TechNet and MSDN. Retail owners of Windows 8 and 8.1 could upgrade for free through the Windows Store, and Windows 7 users could use Windows Update. Along with extra test builds of Windows 10, which are only available to Windows Insiders, Windows 10 is continuously updated and made available to users at no additional cost. Enterprise-level devices may receive these updates more slowly.
